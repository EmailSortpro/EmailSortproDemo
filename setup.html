1 Créer ou mettre à jour votre App Registration Azure

1. Allez sur Azure App Registrations

2. Cliquez sur "New registration" ou sélectionnez une application existante

3. Si vous créez une nouvelle app :

  • Nom : EmailSortPro
  • Supported account types : Accounts in any organizational directory (Any Azure AD directory - Multitenant) and personal Microsoft accounts
  • Redirect URI : Sélectionnez "Single-page application (SPA)"

2 Configurer les URI de redirection (CRITIQUE)

Type de plateforme : Single-page application (SPA)

3 Configurer les permissions API

1. Dans votre app Azure, allez dans "API permissions"

2. Cliquez sur "Add a permission" → "Microsoft Graph" → "Delegated permissions"

3. Ajoutez ces permissions :

  • ✓ User.Read - Lire le profil utilisateur
  • ✓ Mail.Read - Lire les emails
  • ✓ Mail.ReadWrite - Lire et modifier les emails

4. Cliquez sur "Add permissions"

5. Optionnel : Cliquez sur "Grant admin consent" pour éviter le consentement individuel
